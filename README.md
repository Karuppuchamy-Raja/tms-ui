# TMS Website

A web application for Tamilnadu Muthtamizh Sangam built with React

## Tech Stack

- React
- TailwindCSS
- React Query
- React Router DOM

## Setup Instructions

1. Clone the repository
```bash
git clone https://github.com/Karuppuchamy-Raja/tms-ui.git
cd tms-website
```

2. Install dependencies
```bash
# Install TailwindCSS and its dependencies
npm install -D tailwindcss@3 autoprefixer postcss

# Initialize TailwindCSS configuration
npx tailwindcss init -p

# Install dependencies
npm i
```

3. Start the development server
```bash
npm run dev
```
